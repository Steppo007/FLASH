# SVG Components

For including SVGs in your React app as React components, we recommend using the library [SVGR](https://react-svgr.com/). 
If you are using Vite, the easiest way to use it is by using the plugin [vite-plugin-svgr](https://www.npmjs.com/package/vite-plugin-svgr).

```bash
npm install -D vite-plugin-svgr
```

Register the plugin in your `vite.config.js`:

```javascript
import svgr from 'vite-plugin-svgr';

export default {
  // ...
  plugins: [svgr()],
};
```

To silence TypeScript errors like `Cannot find module...` when importing, you can add the following to your vite declaration file (`vite-env.d.ts`):

```typescript
// ...
/// <reference types="vite-plugin-svgr/client" />
```

For details and information about plugin configuration see the [documentation](https://www.npmjs.com/package/vite-plugin-svgr) for `vite-plugin-svgr`.



# Example Usage

```jsx
// the following import works with vite and vite-plugin-svgr
import NotificationBell from '@flash/ui-components/assets/svgs/icons/notificationBell.svg?react';

// set the color directly
<NotificationBell width={50} height={50} color="black" />;

// allow the svg to inherit the text color
<div style={{ color: 'black' }}>
  <NotificationBell width={50} height={50} />;
</div>;
```
