### Uses: [PrimeReact Button](https://primereact.org/button)

The KIconButton is a button intended to display a single icon without any label.

<Canvas of={metaData.Story} />
<Controls />

## Usage

You can use the KIconButton almost exactly as you use the KButton. The main differences are:
- the icon button is not allowed to accept a label, and
- the badge for the icon button appears in the corner, not inside the button itself.

## Example:  

The following example demonstrates usage of the KIconButton with an icon, a tooltip, and a badge.
