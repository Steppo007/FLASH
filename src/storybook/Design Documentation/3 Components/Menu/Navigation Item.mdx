## Navigation Item

### Overview

A **Navigation Item** is a streamlined UI element positioned at the upper part of a webpage or application. It houses essential links or menus for seamless and accessible navigation. Navigation items help users quickly access different sections of the site or app.

* * * * *

### When to Use It

-   To provide quick access to essential links or menus.

-   To organize navigation options in a clear and accessible manner.

-   To enhance the user experience by making navigation intuitive and efficient.

#### **Avoid using navigation items when:**

-   The navigation structure is very simple and does not require dedicated navigation items.

-   The navigation items would clutter the interface and overwhelm users with too many options.

-   The navigation is better suited for a different layout, such as a sidebar or footer.

### Variants

-   **Highlighted**: A navigation item that is highlighted to indicate it is currently selected or important.

-   **With Icon**: A navigation item that includes an icon alongside the text for better visual representation.

-   **With Badge**: A navigation item that includes a badge to indicate notifications or updates.

* * * * *

### Behavior

-   **Hover**: The navigation item changes color of background to indicate it is interactive and can be clicked.

-   **Highlighted**: The navigation item appears highlighted to indicate it is currently selected or important.

* * * * *

### Dos and Don'ts

#### **Dos:**

✔ Use navigation items to organize essential links and menus in a clear and accessible manner.\
✔ Ensure navigation items are easily distinguishable by content type and purpose.\
✔ Provide clear and concise labels or icons for each navigation item.

#### **Don'ts:**

✘ Don't overload the navigation bar with too many items, which can overwhelm users.\
✘ Don't rely solely on color to indicate interactivity---use additional visual cues like hover effects.\
✘ Don't use navigation items for actions or commands that are better suited for a different layout, such as a sidebar or footer.
