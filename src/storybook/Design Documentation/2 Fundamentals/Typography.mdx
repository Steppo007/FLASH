# Typography

<blockquote style={{backgroundColor: '#eaf2ff', paddingTop: 'var(--space-2)', paddingBottom: 'var(--space-2)', borderColor: '#357de8', borderRadius: '4px'}}>
Typography plays a key role in ensuring **clarity, consistency, and readability** across Krones products. The Krones.Digital Design System defines a **structured type scale and font tokens** to maintain a **cohesive visual language** while optimizing for usability.
</blockquote>

## Typeface System

-   **Roboto** (for standard UI elements and body text)
-   **Roboto Condensed** (for high-density interfaces and HMI applications)

**Why Roboto?**
-   Optimized for digital screens
-   High legibility at small sizes
-   Modern, neutral, and scalable

<hr/>

## Type Scale System

The Type Scale System establishes a consistent font size hierarchy to maintain visual balance across different UI components.

<img src="images/typography/type-scale-system.png" style={{margin: 'auto'}}/>

**Best Practices**
-   Use larger sizes for hierarchy (e.g., `$type-scale-title1` for main titles).
-   Body text should be at least 16px (`$font-regular`) for accessibility.
-   Keep tooltips and secondary text at 12px+ (`$font-small`) for readability.

<hr/>

## Font Weight System

<img src="images/typography/font-weight-system.png" style={{margin: 'auto'}}/>

**Best Practices**
-   Use bold text sparingly to avoid overwhelming the user.
-   Regular (400) and Medium (500) are preferred for body text.

<hr/>

## Line Height System

<img src="images/typography/line-height-system.png" style={{margin: 'auto'}}/>

**Best Practices**
-   Longer text blocks should use 1.5 line height for improved readability.
-   Headlines and buttons should have a tighter line height (1 - 1.3) for visual balance.
