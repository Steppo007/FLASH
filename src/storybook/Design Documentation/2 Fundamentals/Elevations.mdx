## When to Use Elevation

Elevation should be used intentionally to:

- **Differentiate layers** (e.g., modals, dropdowns, and floating panels).
- **Indicate interactive elements** that require user attention.
- **Enhance readability** by lifting content from the background.

### Best Practices
-   **Use minimal shadows for clarity:** Avoid excessive depth that makes the UI appear cluttered.
-   **Maintain contrast:** Shadows should remain visible in both light and dark themes.
-   **Follow hierarchy:** Higher elevations should be reserved for elements that require more emphasis.
