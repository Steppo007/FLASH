## Icon Usage Guidelines

### Style
-   <span style={{color: '#e06c00', fontWeight: 'bold'}}>⚠ Icons are by default displayed as outlined icons</span>
-   When using Icon-Combinations make sure the icon in the background is outlined and the fron icon is using the filled style

### Size & Scaling
-   Icons should be legible at small sizes (minimum 16x16px).
-   For HMI applications, icons must be scalable without losing clarity.

### Color & Contrast
-   Icons should adhere to Krones' brand colors and contrast well against backgrounds.
-   Avoid using multiple colors within a single icon unless required for data visualization.

### Stroke & Style Consistency
-   Material Design Icons follow Google's guidelines (rounded corners, consistent line weights).
-   Krones CC Icons maintain an industrial, structured appearance.

### Labeling & Accessibility
-   Include aria-labels for accessibility.
-   Avoid using icons without text labels unless their meaning is universally recognized.
